# Distrito Streaming para Vercel

Esta carpeta es una copia independiente lista para publicar en Vercel.

## Cuentas nuevas recomendadas

Para no usar cuentas que ya tienes abiertas:

1. Crea un correo nuevo para el proyecto, por ejemplo `distritostreaming.admin@gmail.com`.
2. Crea una cuenta nueva en GitHub con ese correo.
3. Crea una cuenta nueva en Vercel con ese mismo correo.
4. Sube esta carpeta a un repositorio nuevo.
5. En Vercel, importa ese repositorio y publica.

## Proyecto

- Entrada principal: `index.html`
- Assets: `assets/`
- Configuracion Vercel: `vercel.json`

## Usuarios demo locales

Estos usuarios son solo para probar la interfaz local:

- Usuario: `ana@demo.com`
- Administrador: `admin@demo.com`
- Clave demo: cualquier texto en el prototipo actual.

## Importante

Esta version es frontend/prototipo funcional con datos locales del navegador. Para que sea 24/7 real con base de datos, pagos y proveedor automatico, el siguiente paso es conectar:

- Base de datos real.
- Autenticacion real.
- API backend.
- Integracion ProviderService.
- Webhooks de pago.
